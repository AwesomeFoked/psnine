# Psnine
Psnine第三方安卓客户端, 基于React Native、Redux, 支持安卓4.1及以上系统 

# Features
- 支持PSNINE网站的所有功能
- 支持自动签到、消息快捷回复、回复中插入图片、拍摄图片后直接上传、非WIFI环境懒加载图片等功能
- 支持网易云音乐和B站链接的客户端跳转
- 支持主题的切换, 可以切换首页布局和19种主题颜色, 以及对应的强调色和夜间主题

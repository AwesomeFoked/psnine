# Psnine
Psnine第三方安卓客户端, 基于React Native, 支持安卓5.0及以上系统 

# Features
- 支持社区、问答、游戏、排行、约战、基因、圈子、商店、闲游九大类别的显示、主题查看、主题回复
- 支持社区主题的攻略分页、游戏列表、新旧版评论和楼中楼的显示与回复
- 支持登录、签到、用户中心、同步、收藏、用户游戏、留言板和用户圈子
- 支持奖杯列表、奖杯TIP和奖杯评分
- 支持社区、问答、游戏、约战、基因、圈子、商店和闲游八大类别的搜索
